# CARA MENJALANKAN DASHBOARD STREAMLIT

##SETUP ENVIRONMENT
```
1. Install Python
2. Install library yang dibutuhkan (pip install -r requirements.txt) atau install library:
  - streamlit
  - pandas
  - matplotlib
  - seaborn
```

## Menjalankan DASHBOARD
  - Buka terminal
  - Masuk ke dalam folder berisi code
  - jalankan dashboard menggunakan
    (streamlit run dashboard.py)
